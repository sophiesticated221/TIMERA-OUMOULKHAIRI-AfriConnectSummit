# AfriConnect Summit 2026

## Projet de conférence tech panafricaine

Prenom & Nom: Oumoul Khairi TIMERA
Classe: L1 CS 
Année: 2025-2026

## Description du projet

AfriConnect Summit est un site vitrine complet pour une conférence tech fictive panafricaine. 
Le site présente l'événement, le programme, les intervenants et permet l'inscription des participants.

## Technologies utilisées

- HTML5 sémantique
- CSS3 (Flexbox, Grid, Variables CSS, Animations)
- JavaScript vanilla (DOM, Events, IntersectionObserver, localStorage)
- Google Fonts
- Bootstrap Icons

## Fonctionnalités JavaScript implémentées

1. Dark Mode / Light Mode avec persistance localStorage
2. Navbar dynamique (fond/ombre au scroll)
3. Menu hamburger mobile
4. Animations au scroll (IntersectionObserver)
5. Compteurs animés qui s'incrémentent
6. Compte à rebours en temps réel
7. Onglets du programme (3 jours)
8. Filtrage dynamique des intervenants
9. Validation de formulaire
10. Bouton retour en haut
11. Année dynamique dans le footer
